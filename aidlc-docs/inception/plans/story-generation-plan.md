# User Story Generation Plan

## 方針
- **分類方法**: ハイブリッド型（ユーザージャーニー＋機能ベース）
- **詳細度**: 標準（15-25件、明確な受入基準付き）
- **ペルソナ**: シンプル（カジュアルゲーマー1種類）
- **フォーマット**: 「〜として、〜したい。なぜなら〜だから」形式 + 受入基準

## エピック構成
1. **ゲーム開始体験** — タイトル画面から初プレイまで
2. **コアゲームプレイ** — 移動・射撃・敵との戦闘
3. **成長・強化** — レベルアップ・武器・仲間・スキル
4. **ゲーム進行** — ウェーブ・難易度・ボス
5. **ゲーム終了・リプレイ** — ゲームオーバー・スコア・リトライ
6. **クロスプラットフォーム** — PC/モバイル対応
