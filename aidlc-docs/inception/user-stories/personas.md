# ペルソナ定義

## ペルソナ: カジュアルゲーマー（ユウキ）

| 項目 | 内容 |
|------|------|
| **名前** | ユウキ |
| **年齢** | 20代〜30代 |
| **デバイス** | スマートフォン（メイン）、PC（サブ） |
| **プレイスタイル** | 通勤中や休憩時間に5〜15分のショートプレイ |
| **ゲーム経験** | モバイルゲーム（SNS広告で見つけたカジュアルゲーム）を中心にプレイ |
| **モチベーション** | 手軽に爽快感を味わいたい、敵を大量に倒す達成感が好き |
| **技術レベル** | ゲーム操作に慣れているが複雑な操作は苦手、シンプルなUIを好む |
| **目標** | スコア更新、新しい武器やスキルの発見、「あと1回だけ」のリプレイ性 |
| **フラストレーション** | 複雑すぎる操作、長いロード時間、理不尽な難易度 |

### 関連ストーリー
全ユーザーストーリー（US-01〜US-20）
