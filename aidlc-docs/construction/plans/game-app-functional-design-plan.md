# Functional Design Plan - Game App

## 概要
ラストウォー型ディフェンスシューティングゲームの機能設計計画。
ビジネスロジック、ドメインモデル、ビジネスルールを技術非依存で詳細設計する。

## Q&Aサマリー

### Phase 1: ゲームバランスの基本パラメータ
| 質問 | 回答 |
|------|------|
| 敵の移動方向 | 真下に直進（ラストウォー型） |
| XP増加曲線 | 緩やかな指数増加（AI設計） |
| 被ダメージ時の無敵時間 | あり（約1秒） |
| 仲間追加の選択肢表示 | 独立カテゴリ（武器スロットとは別管理） |

### Phase 2: 武器パラメータ詳細
| 質問 | 回答 |
|------|------|
| 武器バランス | 概要指定: 前方=高速発射、拡散=広範囲、貫通=高ダメージ。詳細はAI設計 |
| オート射撃の射程 | 画面全体（射程制限なし） |
