# Unit-01: サウンドシステム Functional Design Plan

**作成日**: 2026-04-10

## 計画ステップ

- [x] ドメインエンティティ定義（AudioContext, SoundNode, BGMSequence等）
- [x] ビジネスルール定義（音量制御、SE同時再生制限、モバイル対応等）
- [x] ビジネスロジックモデル定義（BGM生成、SE生成、再生管理フロー）

## 決定事項

| 項目 | 決定 |
|------|------|
| 連携方式 | 直接呼び出し（AudioManager.playSE()） |
| SE同時再生 | 同種SE最大3〜4音、超過分は無視 |
| 射撃SE | 間引き再生（クールダウン付き、仲間は音量低下） |
